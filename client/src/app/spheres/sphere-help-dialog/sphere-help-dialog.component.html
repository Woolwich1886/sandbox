Слева находятся тогглы, управляющие потоками цветных шаров, которые выбрасывются со случайным номером от 1го до 10ти. Есть кнопка сброса (очистки) которая сбрасывает состояние;

Посередине распологается контейнер где отображаются выпавшие шары;

Cправа находитя лог с тем, что произошло.

Допусловия:

1) если выпало три одноцветных шара, то выпадает черный;

2) если выпало три разноцветных шара, то выпадает шар с цветом последнего и наибольшим числом из тех трёх;

3) если выпал синий и его значение больше последнего выпадавшего красного, то выпадает фиолетовый;

4) при включенном тоггле запускается таймер с автовыпадением, красных-раз в 3 секунды, синих-раз в секунду с 50% вероятности, зелёных по две штуки с вероятностью 25% раз в секунду.
